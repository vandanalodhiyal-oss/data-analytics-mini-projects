# Data cleaning project
This project is based on student marks data.  

# Objective
- Clean raw data (remove duplicates, handle missing values)
- Analyze student performance
- Generate useful insights like average, grade, max & min marks

# Used
- Python
- Pandas
- CSV File

# Output

- Cleaned dataset
- New columns:
- Total
- Average
- Grade
- Max Marks
- Min Marks
